# Cargo Notes

Cargo is Rust's build system and package manager.

## Creating a New Project

```sh
cargo new {project_name}
cd {project_name}
```

## Building and Running

**Build, then Run**

```sh
# Build executable
cargo build

# Run executable
./target/debug/{project_name}[.exe]
```

**Build and Run**

```sh
cargo run
```
