# Book That App Notes

BookThatApp (BTA) is a Shopify app that allows booking of a Shopify store's products.


## Uncategorized

- BTA features 6 different booking forms.
- Allows configuration of
	+ Resources
	+ Locations
	+ Zip Codes
	+ Staff Accounts
	+ Templates
	+ Opening Hours
	+ Booking Fields
	+ Shipping Methods
	+ Reports
	+ Widgets
